# Proposal: Optimización de Impresión A4 para 5 Semanas

## Intent
Ajustar las medidas verticales, tipografía, márgenes y relleno del documento PDF/imprimible para garantizar que quepan **al menos 5 semanas completas** en 1 sola hoja A4 sin desbordar a una segunda página.

## Scope
- Frontend: `asignacion-list.component.ts` (plantilla de impresión, estilos CSS `@media print`, `@page` y lógica de visualización).
