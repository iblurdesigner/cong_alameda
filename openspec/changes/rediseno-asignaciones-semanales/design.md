# Design: Rediseño de Asignaciones Unificadas por Semana

## Formato de Tabla Unificada por Semana
En lugar de dividir la tabla en columnas `MIÉRCOLES` y `SÁBADO`, la estructura por semana seguirá el formato oficial de la congregación:

```
+------------------------------------------------------+
| Función               | Asignación                   |
+------------------------------------------------------+
| Fecha:                | Semana del 20 al 26 JULIO    |
| Presidente:           | Alejandro Herrera            |
| Lector Atalaya:       | Christian Trujillo           |
| Acomodadores:         | César Pancho / Vinicio Ortiz |
| Plataforma:           | Pablo Panchi                 |
| Mic. Derecho:         | Isaías Erazo                 |
| Mic. Izquierdo:       | Hernán Varas                 |
| Parqueadero:          | Olger Flores                 |
| Aseo del Salón:       | Grupo 1 - Centro             |
+------------------------------------------------------+
```

## Beneficios
1. **Zero Redundancia**: El usuario asigna la persona/grupo una sola vez por semana.
2. **Máxima Eficiencia Espacial**: Cada bloque semanal requiere solo 2 columnas. En una hoja A4 caben holgadamente de 4 a 5 semanas completas en una sola página.
